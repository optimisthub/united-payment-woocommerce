Payment plugin for United Payment for WooCommerce - Accept payments in Georgia (GEL) with United Payment integration.

== Description ==

Payment plugin for United Payment that enables you to accept credit and debit card payments in Georgian Lari (GEL) on your WooCommerce store.

* Secure payment processing
* Test mode for development
* WooCommerce HPOS and Blocks support
* Automatic order status updates
* Transaction logging

== Installation ==

1. Install and activate the plugin from the WordPress Plugins screen.
2. Navigate to WooCommerce → Settings → Payments and enable **United Payment**.
3. Select **Manage** and enter your Dealer Code, API Username, and API Password from United Payment.
4. Use **Test mode** while verifying your checkout flow, then disable it for live payments.
